ขั้นตอนการรัน Script  

1) เปิดโฟลเดอร์ EmpeoAutomated ใน vscode  

2) เปิด new terminal  

3) รันคำสั่ง       python -m venv .venv

4) รันคำสั่ง  			.\.venv\Scripts\Activate.ps1 

5) รันคำสั่ง       pip install PyYAML 

6) รันคำสั่ง       pip install robotframework-seleniumlibrary

7) รันคำสั่ง 			robot ./testcase/RegisterTestcase.robot

8) โปรแกรมจะแสดงหน้าwebEmpeo และรันคำสั่งตามสคริปที่เขียนไว้
